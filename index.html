Frequently asked questions

Syncless Chat works by recognizing the number you're chatting with on WhatsApp Web and showing you the CRM contact with that phone number, so you can create new linked notes and tasks.

If it doesn't find a contact with that phone number, it will let you create a new one.

To install Syncless Chat, you need to

  1. Install the app from your CRM Marketplace (Less Annoying CRM).
  2. Install the Chrome Extension for WhatsApp Web.

Once you installed the app correctly, you will see the Syncless Chat sidebar when you use WhatsApp Web.

Syncless Chat allows you to select specific messages you want to save as a note to your CRM contact.
To do that you can click on "Select messages", select the ones you want to export, and then "Create".
This will create a note in your CRM attached to the contact that you are chatting.

For Syncless Chat to work, you need to enable cookies on your browser. You can do that inside the settings page (Settings -> Privacy and security -> Cookies and other site data).

You can choose one of the following options:

  • Allow all cookies
  • Block third-party cookies in incognito
  • Manually add the url synclesschat.com under "Sites that can always use cookies"
